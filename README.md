# Initial Setup
created base npm directory and ran the following:
```
npm init -y
npm install -D typescript vite vue @vitejs/plugin-vue
```

```
mkdir -p packages/ui
mkdir -p packages/theme-default
mkdir -p packages/theme-dark
mkdir -p apps/storybook
```

```
cd packages/ui
npm init -y
npm install vue
npm install -D vite vue-tsc
```
